# Conceptos — Errores de Programación

## ¿Qué es un error numérico?
Un error numérico es la diferencia entre el valor exacto de una cantidad
y el valor aproximado que obtiene una computadora al representarla o
calcularla. Surgen por las limitaciones físicas del hardware y del
sistema de numeración binario.

## Tipos de error

### Error absoluto
Diferencia directa entre el valor real y el valor aproximado.
> Error absoluto = |valor real - valor aproximado|

### Error relativo
Porcentaje de qué tan grande es el error respecto al valor real.
> Error relativo = |valor real - valor aproximado| / |valor real|

### Error de truncamiento
Ocurre cuando se corta una operación matemática infinita
(como una serie) en un número finito de términos.

### Error de redondeo
Ocurre cuando la computadora no puede representar exactamente
un número en binario y lo aproxima al más cercano representable.
