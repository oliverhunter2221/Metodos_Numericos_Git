# Conceptos — Métodos de Solución de Ecuaciones

## ¿Qué es una ecuación no lineal?
Una ecuación no lineal es aquella donde la incógnita aparece con
exponentes, dentro de funciones trigonométricas, logarítmicas o
exponenciales. No se puede despejar directamente como en una
ecuación lineal, por eso se usan métodos numéricos iterativos
para aproximar su solución.

## Conceptos clave

### Raíz de una función
Es el valor de x donde f(x) = 0. Encontrar la raíz es el objetivo
de todos los métodos de este tema.

### Iteración
Proceso de repetir un cálculo usando el resultado anterior como
punto de partida, acercándose cada vez más a la solución exacta.

### Convergencia
Un método converge cuando las aproximaciones sucesivas se acercan
cada vez más a la solución real.

### Divergencia
Un método diverge cuando las aproximaciones se alejan de la
solución en lugar de acercarse.

### Tolerancia
Valor mínimo aceptable del error. Cuando el error es menor que
la tolerancia, se considera que el método encontró la solución.

### Intervalo [a, b]
Rango de valores donde se busca la raíz. Para algunos métodos
es necesario que f(a) y f(b) tengan signos opuestos, lo que
garantiza que existe una raíz en ese intervalo.
